# 【超初心者向け】iPhoneアプリ公開マニュアル (Windows → Mac)

このマニュアルは、**「Macやプログラミングに詳しくない方」**でも迷わず操作できるように、すべての手順を１から丁寧に解説しています。
Windowsでの開発は終わりました。ここからは**Mac**を使って、アプリをAppleに提出する作業を行います。

---

## 📋 準備するものリスト

作業を始める前に、手元にこれがあるか確認してください。

1. **Mac本体** (MacBookなど)
2. **Apple ID** (普段iPhoneで使っているものでOK)
3. **クレジットカード** (Apple Developer Programの登録に必要です)
4. **iPhone** (実機テスト用。なくても大丈夫ですが、あると安心です)

---

## ステップ 1: Appleの開発者アカウントを作る

iPhoneアプリを公開するには、Appleに「開発者」として登録し、年会費（約15,000円）を支払う必要があります。これは避けて通れません。

1. Macのブラウザ（SafariやChrome）で [Apple Developer Program](https://developer.apple.com/jp/programs/) にアクセスします。
2. **「登録」** ボタンをクリックします。
3. 画面の指示に従って、Apple IDでログインし、個人情報を入力して支払いを済ませてください。
    * ※ 支払いが完了してから、Appleの承認が下りるまで **最大48時間** かかることがあります。承認メールが届くまで待ちましょう。

---

## ステップ 2: Macに「Xcode」を入れる

「Xcode（エックスコード）」は、Appleが提供しているアプリ作成ツールです。これがないとiPhoneアプリは作れません。

1. Macの **「App Store」** アプリを開きます（青い「A」のアイコン）。
2. 検索窓に「Xcode」と入力して検索します。
3. **「入手」** またはクラウドのマークを押してインストールします。
    * ※ サイズが非常に大きい（数GB）ので、Wi-Fi環境で1時間〜数時間かかることがあります。気長に待ちましょう。
4. インストールが終わったら、一度Xcodeを開いて、「Agree（同意する）」などのボタンを押して初期設定を済ませておいてください。

---

## ステップ 3: Macにコードを持ってくる

Windowsで作ったプログラムを、Macにコピーします。

1. Macで **「ターミナル」** というアプリを開きます。
    * ※ 見つからない場合は、キーボードの `Command (⌘)` キーを押しながら `Space` キーを押し、出てきた検索窓に「ターミナル」と入力してEnterを押してください。黒か白の画面が出てきます。

2. 以下のコマンド（魔法の言葉）をコピーして、ターミナルに貼り付け、Enterキーを押してください。
    * これは「デスクトップに移動する」という命令です。

    ```bash
    cd Desktop
    ```

3. 次に、以下のコマンドをコピーして貼り付け、Enterキーを押してください。
    * これは「GitHubから最新のプログラムをダウンロードする」という命令です。

    ```bash
    git clone https://github.com/hiroto-1020/AISocialManager.git
    ```

4. デスクトップに `AISocialManager` というフォルダができているはずです！確認してください。

---

## ステップ 4: アプリの準備をする

ここからは、ダウンロードしたプログラムを「iPhoneアプリ」の形に変換します。

1. ターミナルで、以下のコマンドを実行します。
    * 「ダウンロードしたフォルダの中に入る」という命令です。

    ```bash
    cd AISocialManager
    ```

2. 次に、以下のコマンドを実行します。
    * 「必要な部品をインストールする」という命令です。少し時間がかかります。

    ```bash
    npm install --legacy-peer-deps
    ```

3. 次に、以下のコマンドを実行します。
    * 「アプリのビルド（組み立て）」を行う命令です。

    ```bash
    npx cap sync
    ```

4. 最後に、以下のコマンドを実行します。
    * 「Xcodeを開く」という命令です。

    ```bash
    npx cap open ios
    ```

    * これで、自動的にXcodeが立ち上がり、アプリのプロジェクトが開きます。

---

## ステップ 5: Xcodeでの設定 (署名)

ここが一番の難関ですが、落ち着いてやれば大丈夫です。「署名（Signing）」という設定をします。

1. Xcodeの左側のリストの一番上にある **「App」** (青いアイコン) をクリックします。
2. 右側の画面で **「Signing & Capabilities」** というタブをクリックします。
3. **「Team」** というところがおそらく「None」や赤くなっていると思います。ここをクリックして、**「Add an Account...」** を選びます。
4. Apple IDとパスワードを入力してログインします。
5. ログインできたら、Teamのところで自分の名前（Personal Team）を選びます。
6. その下の **「Bundle Identifier」** というところが `com.example.app` のようになっている場合は、世界で唯一の名前に書き換える必要があります。
    * 例: `com.yourname.aisocialmanager` (yournameは自分の名前に変えてください)
    * ※ これを変えたら、今後ずっとこの名前を使います。メモしておきましょう。

---

## ステップ 6: アプリをAppleにアップロードする

いよいよアップロードです！

1. Xcodeの画面一番上、真ん中あたりに `Any iOS Device (arm64)` と書いてある場所がありますか？
    * もし `iPhone 15 Pro` などの機種名になっていたら、そこをクリックして **「Any iOS Device (arm64)」** を選んでください。

2. 画面一番上のメニューバー（リンゴマークの横）から **「Product」** をクリックし、その中の **「Archive」** をクリックします。
    * ビルドが始まります。パソコンが頑張って計算するので、数分待ちます。

3. 完了すると、「Archives」という新しいウィンドウが開きます。

4. 右側にある青いボタン **「Distribute App」** をクリックします。

5. あとは基本的に「Next」や「Upload」を選んで進んでいけばOKです。
    * Method of distribution: **App Store Connect** を選択 → Next
    * Destination: **Upload** を選択 → Next
    * いろいろチェックボックスが出ますが、そのまま Next
    * 最後に **「Upload」** ボタンを押します。

6. 「Successfully uploaded」と表示されたら、Macでの作業は完了です！お疲れ様でした！

---

## ステップ 7: 最後の仕上げ (Webブラウザ)

最後に、アプリの説明文などを設定して審査に出します。

1. [App Store Connect](https://appstoreconnect.apple.com/) にアクセスしてログインします。
2. **「マイApp」** をクリックします。
3. アップロードしたアプリが表示されているはずです（処理中でまだ表示されない場合は、15分〜30分ほど待ってください）。
4. 以下の情報を入力していきます。
    * **スクリーンショット**: iPhoneでアプリの画面を撮った画像をアップロードします（最低3枚くらい）。
    * **プロモーション用テキスト**: アプリのキャッチコピーなど。
    * **概要**: アプリの詳しい説明文。
    * **キーワード**: 検索されそうな言葉（例: SNS, 自動化, AI, 運用）。
    * **サポートURL**: `https://ai-social-manager-lilac.vercel.app` (あなたのWebサイトのURL)
    * **マーケティングURL**: 同上
    * **プライバシーポリシーURL**: `https://ai-social-manager-lilac.vercel.app/privacy`
5. すべて入力したら、右上の **「審査へ提出」** をクリックします。
